Most phylogenetics software tools are distributed as command-line applications. Some extra guidance on how to install and run command-line based programs on Microsoft Windows can be found here. (Linux users probably need no help with that, and since I don't have a mac PC at my disposal, the apple people will have to google a bit).

Code examples are in the julia programming language, which is a great language for scientific computing. Even if you are unfamiliar with Julia, most code examples should be easy to read and readily translated to your favorite programming language, which would be a good exercise. If you want to follow along in julia, download the latest julia version for your operating system from https://julialang.org/. If you encounter using <...> statements, this imports external julia packages. To install a julia package, open the REPL (by typing julia at the command line for instance), type ] (now you've entered the package manager pkg) and type add package.
